# Automação de Testes
Projeto definido com todas as propriedades necessárias para automação de testes via browser utilizando Selenium WebDriver e JUnit

Este Projeto Depende dos seguintes configurações:

Maven
Selenium
Junit
Hamcrest
ChromeDriver
Gecko Driver
